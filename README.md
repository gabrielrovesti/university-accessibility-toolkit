## ⚡ Metrica dei 30 Secondi

**Regola operativa**: Se NVDA impiega >30 secondi per far comprendere una domanda, applicare gli esempi di questo toolkit.

### Test Workflow:
1. Apri file con NVDA
2. Naviga domanda con tasti H/T/G
3. Cronometra comprensione
4. Se >30 sec → usa esempio corrispondente dalla cartella 3
